# Chat App!

## tutorial :</h2>

    You can use this chat in the following way:
    First you must register or log in, in order to enter the chat app.
    In the lobby you can choose to create a room or enter one.
    Once a user is in a room he can send messages for everyone in the room to see.


## Usage :

   bash Chat-Project\auto.sh  -  to display usage information.
   bash Chat-Project\scripts\init.sh <Version>   -  to build and run the app locally. 
    
   In your web browser enter the address : http://localhost:5000
